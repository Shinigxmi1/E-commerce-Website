# Comforty E-commerce App

Welcome to **Comforty E-commerce App**. This is a modern, fully responsive e-commerce application built with React, Tailwind CSS, and other cool libraries.

---

## Project Overview

This project features:

- **Auto Layout**: Responsive across different screen sizes.
- **Technologies Used**:
  - **React**: For the UI components.
  - **Tailwind CSS**: For styling.
  - **React Router**: For routing.
  - **Lucide-React**: For icons.
  - **React Slick Slider**: For the carousel slider.

It's a modern e-commerce app with a smooth UX/UI design.

---

## Installation Guide

### 1. **Create a React JS Template with Vite**

To start a new React project with Vite, run:

```bash
npm create vite@latest ./ --template react
````

Follow the prompts to complete the setup.

### 2. **Install NPM Package Manager**

To install the required packages, run:

```bash
npm install
```

Or if you're using a shorthand, you can run:

```bash
npm i
```

### 3. **Install React Router**

For routing in your React app, use:

```bash
npm install react-router
```

### 4. **Install Lucide-React (Icons)**

To add icons to your project:

```bash
npm install lucide-react
```

### 5. **Install Tailwind CSS**

To install Tailwind CSS, run:

```bash
npm install tailwindcss @tailwindcss/vite
```

After this, visit the [Tailwind CSS documentation](https://tailwindcss.com/docs) for further setup details.

### 6. **Install React Slick Slider**

For the carousel slider, install:

```bash
npm install react-slick slick-carousel
```
